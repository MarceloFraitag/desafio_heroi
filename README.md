# 🦸‍♂️ Desafio: Classificador de Nível de Herói

Olá! Este é um dos projetos que estou desenvolvendo durante minha jornada de aprendizado na **DIO**. Como atuo como IT Support Analyst e System Administrator, gosto de focar em criar soluções que, além de cumprirem o objetivo, sejam limpas e organizadas.

### 📝 Sobre o que é este projeto
Neste desafio, apliquei conceitos fundamentais de lógica de programação para criar um classificador de níveis baseado em experiência (XP). Para deixar o exercício mais interessante e prático, utilizei uma lógica inspirada no sistema de elos do **League of Legends** e adicionei personagens icônicos de animes como parte da massa de dados.

### 💻 Tecnologias que utilizei
- **JavaScript**: A base da lógica.
- **Node.js**: Para execução do ambiente.
- **Git/GitHub**: Para versionamento e organização do meu portfólio.
